아르케스파 소개
Brand Introduction

Beginning 시작, Origin 근본, Source of element 근원

도심속 웰니스 스파, 아르케스파

아르케스파는 단순한 피부 건강을 넘어 '건강한 아름다움'을 추구합니다.

아르케스파의 모든 프로그램은 일시적인 효과가 아닌 근본적인
몸의 문제를 찾고, 최적의 솔루션을 제안합니다.

문제점 도출
Ldentifying Problems

충분히 좋은 디자인이지만 더 좋은 디자인을 하기 위해
이용자의 시선으로 바라보고 분석하였습니다.

꼼꼼히 사이트를 분석하고총 3가지 개선점을 제안합니다.

01 좁은 헤더의 영역

저의 개인적인 생각으로 헤더의 영역이 넓은 편이 아닌데 영역에 비하여
로고가 크다는 생각이 들었습니다. 그래서 로고의 위아래 여백이 없어서
시각적으로 조금 불안정한 느낌을 받았습니다. 로고의 크기를 줄이면
너무 작아질 것 같아서 헤더의 영역을 조금 늘려서 여백을 주는 것이
좋을 것 같다는 생각이 들었습니다.

헤더의 영역이 좁고 위 아래의 여백이 부족하여,
시각적으로 불안정한 느낌을 받았습니다.
헤더의 영역을 늘리는 것도 좋은
방법이라고 생각됩니다.

02 메인 배너

저는 사이트를 방문하면 어떤 브랜드 있는지 어떤 상품이 있는지 메인 페이지
배너 내의 내용으로 미리 브랜드의 분위기와 제품을 파악하는 편입니다.
메인 배너는미적인 요소 뿐 아니라 이처럼 정보를 전달하는 중요한
역할을 하기 때문에 추가 하였습니다.

배너의 역할은 미적인 요소 뿐 아니라 정보를
전달하는 역할을 하기 때문에 컨텐츠에
추가 하였습니다.

03 접근성 &편의성

이용자의 편의성과 접근성을 고려해봤을 때 메인 페이지에 브랜드에 대한
소개, 어떤 프로그램이 있는지, 예약 방법 등의 컨텐츠를 많이 찾거나 정보의
중요도 순으로 배치하면 원하는 정보를 쉽게 얻을 수 있어 특히 중년층의
이용자가 편리할 것 같고 접근성 측면서도 좋을같습니다.

컨텐츠를 추가하여 원하는 정보를 쉽게 얻을 수
있게 하는 것이 다양한 이용자의 연령층의
고려했을 때 편의성과 접근성 측면에서
더욱 좋을 것 같습니다.

Solution

고객이 한눈에 브랜드와 상품에 대한 정보를 얻을 수 있도록 메인배너를 배치하고, 이용자들의 편의성과 중년층 고객을 배려하여
원하는 정보를 쉽게 얻을 수 있록 해당 브랜드의 소개, 프로그램 소개, 예약&상담, FAQ등의 컨텐츠를 메인페이지에
배치하였습니다. 특히 문의 상담 & 예약하기 컨텐츠에는 이미지를 호버하면 해당 예약사이트로
연결할 수 있는 버튼을 넣었습니다.

원하는 정보를 쉽게 얻을 수 있고 접근성과 편의성을
고려하여 다양한 정보를 담은 배너와 컨텐츠를
추가하고, 컨텐츠 크기를 키워 시각적으로
느껴지는 답답함을 해결하였습니다.
